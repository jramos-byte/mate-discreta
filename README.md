# Ruta Óptima para Ciudades de Japón Mediante Grafos Ponderados
Este repositorio contiene nuestro proyecto que tiene como objetivo el modelamiento matemático y la implementación computacional de una red de transporte interurbana para optimizar trayectos utilizando teoría de grafos

# Integrantes

# Descripcion del proyecto
Nuestro proyecto consiste en modelar una red de ciudades mediante un grafo,
para la cual creamos una aplicación en Pyton que nos permite calcular las rutas 
óptimas entre 15 ciudades de Japon, la cual establece conexiones reales entre ellas 
utilizando como peso la distancia, tiempo de viaje o costo estimado,
La cual utiliza información proviente de fuentes verificables que respaldan las conexiones
de las rutas establecidas
