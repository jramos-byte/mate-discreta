Nuestro proyecto consiste en modelar una red de ciudades mediante un grafo
para la cual usamos una aplicación creada en Pyton que permite calcular la rutas 
óptimas entre 15 ciudades de Japon, la cual establece conexiones reales entre ellas 
utilizando como peso la distancia, tiempo deviaje o costo estimado,
La cual utiliza información proviente de fuentes verificables que respaldan lasconexiones
de las rutas establecidas
